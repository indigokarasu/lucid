# lucid

<p align="center">
  <img src="./assets/readme/hero.jpg" width="100%" alt="Lucid: nightly journal curator — batch-processes skill journals into the memory store">
</p>

Lucid batch-processes OCAS skill journals each night. It classifies entries for filing into the memory store — as knowledge graph triples, behavioral rules, or skippable noise — and produces journals that downstream skills can act on.

**Capabilities:**
- Nightly journal scanning across all skills
- Entry classification: triple, rule, signal, or skip
- Memory store write-through with provenance tagging
